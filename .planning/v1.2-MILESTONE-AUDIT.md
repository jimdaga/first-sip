### Integration Issues (Phase 18/19 boundaries)

1. **Markdown rendering gap** — `extractContent()` in `dashboard/viewmodel.go:366` writes raw Markdown into `<p>` tags. Phase 18 must add Markdown-to-HTML conversion (affects GEN-02, GEN-04).

2. **Dual generation paths** — Legacy `briefing:generate` webhook pipeline coexists with new `plugin:execute` CrewAI pipeline. Two parallel paths with different data models (`Briefing` vs `PluginRun`). Phase 18 must use the plugin path; Phase 19 removes the webhook path (affects CLN-01, CLN-02, GEN-01, GEN-04).

3. **Compile-time dependency** — `briefing.templ` `renderContent()` imports `webhook.BriefingContent`. Removing the `webhook` package (CLN-02) requires updating templates first (affects CLN-02).
